Long Methods are methods that span too many lines. When a method gets too long, perhaps it is responsible for more tasks than it should be.
~~~ python
def long_method(self):
    # Do first thing
    
    # Do second thing
    
    # Do third thing
    
    # Do fourth thing
~~~
